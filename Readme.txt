Файл для запуска сервера album_server.py Запускать из командной строки, предварительно перейдя в каталог с файлом, или в редакторе. Поиск альбома производится в браузере: например http://localhost:8080/albums/Pink Floyd
В файле album.py код для работы с базой данных (импортируется в album_server.py)

Пример POST запроса на сервер из командной строки:
http -f POST localhost:8080/albums year="1967" artist="New Artist" genre="Rock" album="Super"

Производится проверка на правильность ввода года выпуска альбома и наличие альбома указанного артиста в базе данных
